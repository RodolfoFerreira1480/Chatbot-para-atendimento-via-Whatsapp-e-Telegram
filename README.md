# Chatbot-para-atendimento-via-Whatsapp-e-Telegram
Bot multi-plataforma (WhatsApp e Telegram) integrado à API do Gemini usando Node.js. Desenvolvido com Express, Axios e Telegraf, o sistema atende clientes de forma automatizada com IA, tratando instabilidades de rede (erro 503) e formatação do nono dígito de forma totalmente resiliente.
